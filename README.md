# Mouse Click Counter Panel
Basic Frontend JavaScript Project

## Overview
It’s a simple, interactive frontend web project that demonstrates basic DOM manipulation and event handling in JavaScript, providing users with the ability to increase, decrease, and reset a numeric counter.
The project also supports keyboard interactions using the arrow keys.

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)

## Features
- Increment counter using the "+" button or ArrowUp key
- Decrement counter using the "-" button or ArrowDown key (with non-negative values)
- Reset counter using the "Reset" button
- Responsive and visually styled interface
- Hover effects for buttons for improved user experience

## Installation
1. Clone the repository using: git clone https://github.com/dmitry-backend/mouse-click-counter-panel.git
2. Navigate to the project directory using: cd mouse-click-counter-panel
3. Open index.html in your web browser

## Running the Application
Simply open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).
No backend or server setup is required.

## Usage
- Click the "+" button or press the ArrowUp key to increase the counter
- Click the "-" button or press the ArrowDown key to decrease the counter (cannot go below 0)
- Click the "Reset" button to reset the counter to 0
